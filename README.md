wfinal-utilities
===========

基于jfinal框架的项目基础包

目的是为了更加简化Jfinal开发流程，定义通用的controller和db

关于框架的详细说明，请参见[wiki](https://github.com/gefangshuai/wfinal-utilities/wiki)

**有任何疑问请在issues中留言！**
# Thanks
- java: 1.7+
- [jfinal](http://jfinal.com): 2.0
- jfinal-ext: 3.1.2
