wfinal-core
===========

基于jfinal框架的项目基础包

目的是为了更加简化Jfinal开发流程，定义通用的controller和db

# Thanks
- java: 1.7+
- [jfinal](http://jfinal.com): 1.9+
- jfinal-ext: 3.1.2
